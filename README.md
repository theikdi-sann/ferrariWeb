# Ferrari Brand Experience

A modernized, immersive web experience for Ferrari, featuring a real-time 3D car configurator, racing hub, and genuine parts catalogue.

## Features
*   **3D Configurator**: Customize road cars with paint, rims, and interior options.
*   **Racing Hub**: Explore the SF-23 and 296 GT3 with technical specifications.
*   **Collections**: Browse genuine Ferrari parts and technical gear.
*   **Experiences**: Discover the history and passion of the brand.

## Quick Start
This project requires a local server to load 3D assets correctly (due to CORS policies).

1.  Open your terminal in the project directory.
2.  Run the following command:
    ```bash
    python3 -m http.server
    ```
3.  Open your browser and navigate to `http://localhost:8000`.
